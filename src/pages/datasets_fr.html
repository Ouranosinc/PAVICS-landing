Info! NetCDF Markup Language (NcML): Les jeux de données PAVICS sont souvent fournis par l'entremise d'agrégations NcML, une manière pratique de combiner de multiples fichiers en un seul jeu de données logique. Par exemple, la version 1.0 de l'ensemble standard des scénarios climatiques ajustés d'Ouranos (cb-oura-1.0) contient 22 jeux de données climatiques, où chaque simulation consiste en une agrégation NcML. Du point de vue de l'utilisateur, chaque simulation de cb-oura-1.0 est accessible par un seul fichier .ncml, alors qu'en réalité celle-ci est une agrégation de fichiers annuels individuels pour chacune des trois variables (tasmin, tasmax, pr), sur la période 1950-2100.
